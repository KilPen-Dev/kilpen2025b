# KilPen Technical Services Website

Corporate website for KilPen Technical Services - IT with Aloha.

## Tech Stack

- [Hugo](https://gohugo.io/) - Static site generator
- [Hugoplate](https://github.com/zeon-studio/hugoplate) - Theme
- [TailwindCSS](https://tailwindcss.com/) - Styling

## Development

```bash
# Install dependencies
npm install

# Run development server
hugo server

# Build for production
hugo --gc --minify
```

## Deployment

Site is deployed via GitHub Pages at [kilpen.com](https://kilpen.com)
